Telco Benefit Wallet – Token Economy

Visual diagrams for token economy, app architecture, and customer journey.

1. Token Economy / Token Lifecycle

High-level flow of how user actions become tokens, perks, tiers and retention.

flowchart TD A[User actions] --> B[Token engine] B --> C[Tokens earned] B --> D[Perks unlocked] B --> E[Tier progression] C --> F[Wallet balance] D --> G[Perk cards] E --> H[Tier dashboard] F --> I[Telco redemption] F --> J[Partner redemption] F --> K[Gifting or trading] I --> L[Higher retention] J --> L K --> L

How the wallet sits inside the app and which sections it contains.

3. Customer Journey – Earn, Spend, Check Status

Simple CJ for getting tokens, spending them and viewing status.

flowchart LR A[Open app] --> B[See wallet teaser] B --> C[Earn tokens] C --> D[Tokens added to wallet] D --> E[Open wallet and redeem] E --> F[Choose benefit] F --> G[Confirm redemption] G --> H[Benefit applied] D --> I[Check status] I --> J[See tier and progress] J --> K[See how to reach next tier] K --> C H --> C
