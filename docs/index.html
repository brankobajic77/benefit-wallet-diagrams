Telco Benefit Wallet – Token Economy

Visual diagrams for token economy, app architecture, and customer journey.

1. Token Economy / Token Lifecycle

High-level flow of how user actions become tokens, perks, tiers and retention.

flowchart TD
    A[User Actions (Purchases, Engagement, Tenure, Referrals)] --> B[Token Engine (Rules and Multipliers)]

    B --> C[Tokens Earned (Liquid Value)]
    B --> D[Perks Unlocked (Trade-in, Accessory Vouchers)]
    B --> E[Tier Progression (Silver, Gold, Platinum)]

    C --> F[Wallet Balance (Visible Tokens)]
    D --> G[Perk Cards in Wallet (Claim or Redeem)]
    E --> H[Tier Dashboard (Status and Benefits)]

    F --> I[Telco Redemption (Bill, Devices, Add-ons, OTT)]
    F --> J[Partner Redemption (Shopping, Travel, Fitness)]
    F --> K[Gifting or Trading (Tokens and Perks)]

    G --> I
    G --> J

    I --> L[Higher Retention (LTV, ARPU, Lower Churn)]
    J --> L
    K --> L

2. App Architecture – Benefit Wallet Module

How the wallet sits inside the app and which sections it contains.

flowchart TD
    A[App Home] --> B[Benefit Wallet]

    B --> C[Balance (Token Count)]
    B --> D[Earn]
    B --> E[Redeem]
    B --> F[My Perks]
    B --> G[My Status]

    D --> D1[Purchases (Bill, Devices, Add-ons)]
    D --> D2[Missions (Engagement Tasks)]
    D --> D3[Referrals]
    D --> D4[Tenure and ARPU Bonuses]

    E --> E1[Telco Benefits (Discounts, Data, Roaming)]
    E --> E2[Partner Offers (Gym, Travel, Shopping)]
    E --> E3[Perks (Trade-in, Accessory Voucher)]

    F --> F1[Active Perks (Valid Now)]
    F --> F2[Upcoming or Expiring Perks]

    G --> G1[Tier Level (Silver, Gold, Platinum)]
    G --> G2[Progress Bar (to Next Tier)]
    G --> G3[Next Rewards (Upcoming Benefits)]

3. Customer Journey – Earn, Spend, Check Status

Simple CJ for getting tokens, spending them and viewing status.

flowchart LR
    A[1. Open App] --> B[2. See Wallet Teaser (Balance + Earn More CTA)]
    B --> C[3. Earn Tokens (Bill Payment, Device Purchase, Mission, Referral)]
    C --> D[4. Tokens Added Instantly to Wallet]

    D --> E[5. Open Wallet and Redeem]
    E --> F[6. Choose: Telco Benefit, Partner Offer, or Perk Card]
    F --> G[7. Confirm Redemption (One Tap)]
    G --> H[8. Benefit Applied (Discount, Data, Voucher)]

    D --> I[Check Status]
    I --> J[My Status Screen (Tier and Progress)]
    J --> K[See How to Reach Next Tier]
    K --> C

    H --> C
