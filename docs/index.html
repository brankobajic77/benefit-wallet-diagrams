Telco Benefit Wallet – Token Economy

Visual diagrams for token economy, app architecture, and customer journey.

1. Token Economy / Token Lifecycle

High-level flow of how user actions become tokens, perks, tiers and retention.

flowchart TD
    A[User Actions
(Purchases, Engagement, Tenure, Referrals)] --> B[Token Engine
(Rules & Multipliers)]

    B --> C[Tokens Earned
(Liquid Value)]
    B --> D[Perks Unlocked
(Trade-in, Accessory Vouchers)]
    B --> E[Tier Progression
(Silver / Gold / Platinum)]

    C --> F[Wallet Balance
(Visible Tokens)]
    D --> G[Perk Cards in Wallet
(Claim / Redeem)]
    E --> H[Tier Dashboard
(Status & Benefits)]

    F --> I[Telco Redemption
(Bill, Devices, Add-ons, OTT)]
    F --> J[Partner Redemption
(Shopping, Travel, Fitness)]
    F --> K[Gifting / Trading
(Tokens & Perks)]

    G --> I
    G --> J

    I --> L[Higher Retention
(LTV, ARPU, Lower Churn)]
    J --> L
    K --> L

3. Customer Journey – Earn, Spend, Check Status

Simple CJ for getting tokens, spending them and viewing status.

flowchart LR
    A[1. Open App] --> B[2. See Wallet Teaser
(Balance + CTA “Earn More”)]
    B --> C[3. Earn Tokens
(Pay bill, Buy device/add-on,
Complete mission, Invite friend)]
    C --> D[4. Tokens Instantly Added
to Wallet Balance]

    D --> E[5. Open Wallet → Redeem]
    E --> F[6. Choose:
• Telco Benefit
• Partner Offer
• Perk Card]
    F --> G[7. Confirm Redemption
(One Tap)]
    G --> H[8. Benefit Applied
(Discount / Add-on / Voucher)]

    D --> I[Check Status]
    I --> J[My Status Screen
(Tier + Progress Bar)]
    J --> K[See “How to Reach Next Tier”
(Required Actions)]
    K --> C

    H --> C
